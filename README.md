# 🤖 Machine Learning Studies

A personal collection of Python scripts built while learning Machine Learning concepts hands-on. This repo grows continuously as I explore new topics and techniques.

---

## 📁 Project Structure
```
Machine-Learning/
│
├── data/                        # Datasets used in experiments
├── utils/                       # Utility/helper functions
│
├── categoricalDataEncoding.py   # Encoding techniques (Label, OneHot, Ordinal...)
├── handleNullData.py            # Handling missing values (imputation, removal...)
└── ...                          # More scripts coming soon!
```

## 🚀 How to Use

1. Clone the repository:
```bash
   git clone https://github.com/kelvidev/Machine-Learning.git
```

2. Install dependencies:
```bash
   pip install -r requirements.txt
```

3. Run any script individually:
```bash
   python dataCleaning.py
```
